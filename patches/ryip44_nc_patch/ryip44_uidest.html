Destinations for Remote Call-In
Input




Help

This window allows the setup of the currently selected input's Remote Call-In destination table.

IX: Up to 20 stations (or 1 group if Target IP is a master station) can be configured by entering the stations' Station Numbers (or a Group Number).

IS: Up to 20 stations, including 1 remote site, can be configured by entering the stations' TermID's.

An additional enable/disable option is available for each station so they can be enabled or disabled individually.

Note: A disabled or blank TermID/Station Number is not added to the SetURI string.
