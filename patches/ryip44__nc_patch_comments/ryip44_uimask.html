Allowed Stations for Output




Help

This window allows you to set up a list of allowed stations for the Door Release function.

Up to 32 stations can be configured by checking or unchecking the corresponding enable flag.

NOTE: If the Linked Stations list on the main page is modified, replacing existing stations' TermID's, enabled flags are always related to the same station position number (1 to 32), so they will automatically reapply to the new TermID.
